# 实 验 题 目

## 1、图片的显示与隐藏

实现如下功能：

点击按钮时，图片消失，按钮上显示文本“显示图片”；再次点击按钮，图片出现，按钮上显示文本“隐藏图片”。

提示：1、ImageView可用isShown()来判断图像是否显示。

​      2、ImageView可用方法setVisibility(View.VISIBLE)设置图片显示，用方法setVisibility(View.INVISIBLE)设置图片消失。
